## BootStrap

부트스트랩은 Frontend FrameWork이다.
해당 프레임워크는 개발자들이 웹사이트를 쉽게 만들 수 있게 도와주는 CSS, Js 기반의 프레임워크이다. 하나의 코드로 휴대폰 태블릿, 데스크탑까지 다양한 기기에서 작동할 수 있게 도와준다.

또한 다양한 기능을 제공하여 사용자가 쉽게 웹사이트를 제작, 유지, 보수 할 수 있도록 도와준다. 

## BootStrap 역사

원래는 Twitter의 내부 개발 도구로 시작되었고, 당시 트위터 개발자였던 Mark Otto, Jacob Thornton이 사내 도구들의 디자인 파편화를 막고 일관성을 유지하기 위해 개발되었다.

- 처음에는 Twitter Blueprint라는 이름으로 불리다가, 오픈 소스로 공개되면서 Bootstrap이라는 이름을 갖게 되었다

- 버전 3.0부터는 모바일 우선주위를 채택하며 반응형 웹 디자인의 표준으로 자리 잡았다.

- 출시 직후 Github에서 가장 인기 있는 프로젝트 중 하나가 되었고, 현재까지도 웹 프론트엔드 개발에서 가장 널리 쓰이는 프레임워크 중 하나이다. 

## BootStrap 장점

- 이미 만들어진 다양한 Componenets

- Community에서 다양하게 생성된 많은 Template들

- 쉽게 전문가 수준의 복잡한 반응형 웹사이트 구축

- 다양한 브라우저들과 호환
