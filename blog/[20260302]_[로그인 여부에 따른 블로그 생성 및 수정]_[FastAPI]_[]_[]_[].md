## 다른 사용자의 악용

get_session_user로 session_user가 있을 경우 블로그 수정 및 삭제가 가능하게 하였다.

다만 해당 블로그 작성자가 아닌 다른 사람이 로그인을 진행하였을 경우 get_session_user 함수를 통과하므로 다른 사람의 글을 마음대로 삭제 또는 수정 또는 생성 하는게 가능하다 

이를 막기 위해서는 get_session_user뿐 만 아니라 계정 권한을 따로 수행할 필요가 있다 

먼저 auth_scv.py에 로직 함수를 만든다
session user랑 해당 블로그 작성자 정보에 따라 False, True를 return 한다. 

```python
def check_valid_auth(session_user: dict, blog_author_id: int, blog_email: id):
    if session_user is None:
        return False
    if ((session_user["id"] == blog_author_id) and (session_user['email'] == blog_email)):
        return True
    return False

```

auth에서 예를 들어 blog를 수정한다고 하자

```python
@router.get("/modify/{id}")
async def update_blog_ui(request: Request, id: int,
                        conn = Depends(context_get_conn), session_user = Depends(auth_svc.get_session_user_prt)):
    
    blog = await blog_svc.get_blog_by_id(conn, id=id)
    is_valid_auth = auth_svc.check_valid_auth(session_user, blog_author_id=blog.author_id, blog_email=blog_email)

    if not is_valid_auth:
        raise HTTPException(status_code=status.HTTP_401_UNAUTHORIZED, detail="해당 서비스는 권한이 없습니다")

```

다만, 애초에 해당 session_user가 블로그 작성자라 다르면 애초에 블로그 수정, 블로그 삭제 버튼 자체가 안보여야 한다. 

auth_svc.get_session_user_ptr과 달리 auth_svc.get_session_user_opt는 session_user가 없으면 None을 발생시키며 로그인이 안되어도, 세션 유저가 블로그 author라 달라도 볼 수 있도록 해야 하므로 is_valid_auth에 대한 HTTPExcpetion은 삭제한다. 

다만, is_valid_auth가 False이면 show_blog.html에서 블로그 삭제, 수정 버튼을 안보이주기 위해서 ui에 is_valid_auth 값을 전달한다 


```python
@router.get("/show/{id}")
async def get_blog_by_id(request: Request, id: int,
                   conn: Connection = Depends(context_get_conn),
                   session_user = Depends(auth_svc.get_session_user_opt)):
    blog = await blog_svc.get_blog_by_id(conn, id)
    blog.content = util.newline_to_br(blog.content)

    is_valid_auth = auth_svc.check_valid_auth(session_user, 
                                              blog_author_id=blog.author_id, 
                                              blog_email=blog.email)

    return templates.TemplateResponse(
        request = request,
        name="show_blog.html",
        context = {"blog": blog,
                   "session_user": session_user,
                   "is_valid_auth": is_valid_auth})

```html
<a href="/blogs" class="btn btn-secondary">홈으로 돌아가기</a>
                {% if is_valid_auth %}
                <a href="/blogs/modify/{{ blog.id }}" class="btn btn-primary">블로그 수정</a>
                <button class="btn btn-danger" onclick="confirmDelete()">블로그 삭제</button>
                {% endif %}

```
